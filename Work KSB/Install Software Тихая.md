Во время экспериментов с тихой установкой мне встречались следующие параметры запуска режима тихой установки:

```
 /silent
 /verysilent
 /quiet
 /qb
 /qn
 /qr
 /passive
 /s
 /S
 /qn REBOOT=ReallySuppress
 /s /v" /qn REBOOT=ReallySuppress
```

Ключи для отмены перезагрузки:

```
 /norestart
 /noreboot
```
